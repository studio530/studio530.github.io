# Studio530

Este proyecto representa la página oficial de Studio530. Diseñada para mostrar servicios, conectar con clientes y crear experiencias únicas.

## Estructura del Proyecto
- **index.html**: Página principal.
- **/css/styles.css**: Estilos personalizados.
- **/js/scripts.js**: Funciones dinámicas en JavaScript.
- **/images/**: Carpeta de imágenes del proyecto.
- **/components/**: Archivos reutilizables (opcional).
- **/fonts/**: Fuentes personalizadas (opcional).

## Requerimientos
1. Navegador moderno.
2. Editor de texto (como Visual Studio Code).
3. Git para clonar el repositorio.
4. Hosting en [GitHub Pages](https://pages.github.com).

#### **Imágenes necesarias**:
1. **Logo**:
   - Archivo: `images/logo.png`
   - Descripción: Logotipo de Studio530.
   - Tamaño sugerido: 200x200 px.

2. **Banner**:
   - Archivo: `images/banner.jpg`
   - Descripción: Imagen de cabecera de bienvenida.
   - Tamaño sugerido: 1920x1080 px.

3. **Servicios**:
   - Archivo: `images/services.jpg`
   - Descripción: Imagen ilustrativa de servicios.
   - Tamaño sugerido: 800x600 px.

#### **Requerimientos adicionales**:
- **Google Fonts**: Fuente "Roboto", enlazada en el `<head>` de `index.html`.
- **GitHub Pages**: Para alojar el sitio.
